## Install

### Clone
```bash
git clone https://github.com/sreedevk/dot ~/.dot
```

### Dependencies

```bash
cd ~/.dot
./scripts/arch_install_deps.sh
```

### Link
```bash
cd ~/.dot
stow stowed
```
